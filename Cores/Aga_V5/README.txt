# Amiga AGA

Core port from Jepalza, ported from Minimig and MIST sources.
Instructions
Put de1_boot.bin and any amiga kickstart (renaming kick.rom) in root SD. 
F12 key for OSD menu.
ScrolLock key to switch VGA/RGB mode, thanks to Neuro.
de1_boot.bin (OSD) spanish version, translate by Edu Arana. 
You can find english version on this link
